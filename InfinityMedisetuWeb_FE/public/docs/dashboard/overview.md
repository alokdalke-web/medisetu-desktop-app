# Dashboard Guide

The Dashboard is your first screen after login. It provides a real-time snapshot of your clinic's operations, tailored to your role. Each user type sees different metrics and panels relevant to their responsibilities.

---

## Admin Dashboard

The admin dashboard gives a complete operational overview of the clinic.

### Key Metrics

| Metric | What It Shows | Action If Declining |
|--------|---------------|---------------------|
| **Total Earnings** | Revenue with % vs previous period | Check avg revenue per visit |
| **Total Appointments** | Booking count with trend | Review marketing or reminders |
| **No-Show Count** | Missed appointments | Review reminder settings |
| **Active Patients** | Patients with recent activity | Check retention efforts |

### Charts & Visualizations

| Chart | What to Look For |
|-------|-----------------|
| **Appointment Status** | High cancellation rate = scheduling issue |
| **Revenue Overview** | Revenue down + appointments steady = lower avg revenue |
| **Patient Growth** | Flat line = acquisition problem |
| **No-Show Trends** | Spikes = specific day/time issue |
| **Symptom Statistics** | Clusters = local outbreak or seasonal trend |

### Live Panels

- **Patient Queue** — Everyone currently checked in, updates in real-time
- **Recent Patients** — Quick-access list for fast follow-up

## Doctor Dashboard

Focused on your personal clinical performance and today's patient flow.

### Key Metrics

| Metric | What It Shows |
|--------|---------------|
| **Total Earnings** | Revenue from your consultations with growth % |
| **Total Appointments** | Your booking count with period comparison |
| **Confirmed Appointments** | Visits confirmed, awaiting patient arrival |
| **Pending Appointments** | Not yet confirmed, may need follow-up |

### Panels

- **Appointment Statistics** — Your completed vs cancelled by month
- **Pending Queue** — Scheduled patients not yet checked in
- **Patient Queue** — Checked-in patients waiting for you (live)
- **Recent Patients** — Quick access to latest consultations

### Call Reception

Use the **Call Reception** button in the header:
- **General Call** — Need front-desk assistance
- **Next Patient** — Ready to see the next patient in queue

---

## Receptionist Dashboard

Built for front-desk situational awareness — what's happening right now.

### Key Panels

| Panel | Purpose |
|-------|---------|
| **Today's Appointments** | Live count of all scheduled visits |
| **Pending Check-ins** | Patients with appointments but not yet arrived |
| **Walk-in Queue** | Patients without appointments, currently waiting |
| **Upcoming Slots** | Next available booking slots across all doctors |
| **Recent Bookings** | Last few appointments created/modified |
| **Doctor Availability** | Which doctors are working and next open slots |

### Quick Actions

- **Create New Appointment** — One-click shortcut
- **Register Walk-in** — Skip menu navigation

### Panels

| Panel | What It Shows |

## Pharmacist Dashboard

Summarizes pharmacy operations at a glance.

### Panels

| Panel | What It Shows |
|-------|---------------|
| **Today's Sales** | Total revenue from pharmacy sales |
| **Low Stock Alerts** | Medicines below minimum threshold |

## Date Filters

All dashboards support date filtering:

| Tab | What It Shows |
|-----|---------------|
| **Today** | Current day's performance |
| **This Week** | Rolling 7-day view |
| **This Month** | Month-to-date vs last month |
| **Custom** | Pick any date range |

> Tip: Period comparison is automatic. "This Month" compares against last month.

- [ ] Check notifications for errors or pending approvals
- [ ] Review low-stock/expiry alerts (pharmacy)
