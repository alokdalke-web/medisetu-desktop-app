# Profile & Settings — Quick Reference

Configure your clinic, personal details, services, availability, and system preferences.

---

## Profile Overview

View and edit your personal details:
- Name, email, phone, qualifications
- Profile photo
- Registration number and specialization (doctors)

---

## Clinic Details

| Field               | Description                                                           |
| ------------------- | --------------------------------------------------------------------- |
| Clinic name         | Displayed on prescriptions and reports                                |
| Address             | Physical location shown to patients                                   |
| Logo                | Appears on prescriptions, reports, printed docs — use high-resolution |
| Operating hours     | When the clinic is open                                               |
| Contact             | Phone, email for the clinic                                           |
| Registration number | Clinic registration/license ID                                        |

---

## Services & Pricing

Manage all consultation services your clinic offers.

| Action         | Steps                                            |
| -------------- | ------------------------------------------------ |
| Add service    | Click "+ Add" → Enter name, duration, fee → Save |
| Edit service   | Click service → Update details → Save            |
| Remove service | Click delete icon → Confirm                      |

> Tip: Updated fees apply to all new bookings immediately.

---

## Doctor Availability

Configure when each doctor is available for appointments.

| Setting          | Description                                |
| ---------------- | ------------------------------------------ |
| Working days     | Days of the week available                 |
| Time slots       | Start and end time per day                 |
| Slot duration    | Length of each appointment                 |
| Buffer time      | Gap between appointments for documentation |
| Max patients/day | Prevents overbooking                       |
| Leaves           | Mark specific dates as unavailable         |

> Important: Set availability completely before enabling patient bookings. Incomplete schedules create gaps that confuse patients.

---

## Medicines

Manage your medicine/composition library for prescriptions.

| Action        | Steps                                                                    |
| ------------- | ------------------------------------------------------------------------ |
| Add single    | Click "+ Add New Medicine" → Enter name, select form → Save              |
| Bulk upload   | Switch to "Bulk Upload" tab → Download template → Fill CSV/XLSX → Upload |
| Edit medicine | Click a medicine row → Update details → Save                             |
| Toggle status | Edit mode → Use the status toggle to enable/disable                      |
| Download list | Click "Download" to export current medicines as text                     |

- Search by brand name or generic composition
- Filter by form type (Tablet, Syrup, etc.)
- Mark frequently prescribed medicines as favorites

## Security & Access

All security settings are consolidated on a single page.

| Feature                    | What It Does                                           |
| -------------------------- | ------------------------------------------------------ |
| Change Password            | Update your login password with strength validation    |
| MFA (Two-Factor)           | Add TOTP code via authenticator app for extra security |
| Login Alerts               | Get notified when someone logs in from a new device    |
| Auto Logout                | Automatically sign out after a period of inactivity    |
| Payment History Visibility | Control whether payment history is visible in the app  |
| Active Sessions            | See all devices logged in (coming soon)                |

> Important: If you see an unrecognized session, terminate it immediately. Enable MFA for admin accounts.

## No-Show Policy

Configure rules for missed appointments:

| Setting         | Options                   |
| --------------- | ------------------------- |
| Auto-tracking   | Enable/disable            |
| Max no-shows    | Threshold before flagging |
| Actions         | Warning → Penalty → Block |
| Cooldown period | Time before count resets  |

## Tips

- Complete clinic details before going live (logo, address, hours)
- Set doctor availability before enabling patient bookings
- Review notification settings — defaults are sensible but worth checking
- Use prescription templates for your top 10 diagnoses
- Change password quarterly and enable MFA for admin accounts
- Payment visibility toggle is in Security & Access — use it to hide financial data from non-admin users
- Use bulk upload for medicines when migrating from another system
- Check active sessions weekly — terminate any you don't recognize
