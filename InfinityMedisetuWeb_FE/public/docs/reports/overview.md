# Reports & Analytics — Quick Reference

Turn accumulated clinic data into actionable insights.

---

## Report Categories

| Category | What It Shows | Key Questions Answered |
|----------|---------------|----------------------|
| **Overview** | High-level performance metrics | "How is the clinic doing overall?" |
| **Appointments** | Status breakdown, doctor distribution, slot utilization | "Are bookings trending up?" |
| **Patients** | Registrations, demographics, visit frequency | "Is my patient base growing?" |
| **Revenue** | Payment mode breakdown, service distribution, trends | "Which services are most profitable?" |
| **Medicines** | Top prescribed, volume trends | "What's prescribed most?" |
| **Staff** | Consultation count, completion rate, feedback | "Who's performing well?" |

---

## Date Range & Comparison

- Set a date range for your analysis period
- Comparison is auto-calculated (same duration, shifted back)
- Example: May 12–18 compares vs May 5–11

| Indicator | Meaning |
|-----------|---------|
| 🟢 Green | Metric improved vs previous period |
| 🔴 Red | Metric declined vs previous period |
| ⚪ Gray | No significant change |

---

## Generating a Report

1. Select report type (from tabs)
2. Set date range
3. Apply filters (doctor, department, payment mode)
4. View on-screen
5. Export as PDF or Excel

---

## Key Metrics Explained

| Metric | Formula |
|--------|---------|
| Completion Rate | Completed ÷ Total Scheduled |
| No-Show Rate | No-Shows ÷ Total Scheduled |
| Avg Revenue/Visit | Total Revenue ÷ Completed Appointments |
| Patient Retention | Returning Patients ÷ Total Active |

---

## Troubleshooting

| Problem | Fix |
|---------|-----|
| No data showing | Expand date range, remove filters, verify data exists |
| Numbers don't match | Check if comparison period is what you expect |
| Revenue seems wrong | Values are in paise internally — verify formatting |

---

## Weekly Review Checklist

- [ ] Appointment completion vs no-show rates
- [ ] Revenue trends (up, down, flat?)
- [ ] Patient feedback scores
- [ ] Subscription usage vs limits
- [ ] Any recurring staff-reported issues
